# 🚀 ZYNGO

ZYNGO is a Full Stack Food Delivery Web Application built with modern web technologies. It provides a complete food ordering and delivery management system with multiple user roles and real-time features.

## ✨ Features

- 🔐 User Authentication & Authorization
- 👤 User Panel
- 🏪 Owner Panel
- 🚴 Delivery Boy Panel
- 🛠️ Admin Panel
- 📍 Real-Time Order Tracking
- 📱 Fully Responsive Design
- 🍔 Restaurant Management
- 📦 Order Management
- 💳 Secure Payment Integration
- ⚡ Fast and Scalable Performance

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Deployment
- Vercel
- Render
